Why Barista

Everything in your menu bar.

Stocks, weather, system stats, calendar, timers, developer tools, sports scores - all live in your menu bar. Click any widget for a rich dropdown.

🎨

Menu Bar Styling

Tint your menu bar with solid colors, gradients, time-based dynamic themes, or frosted glass. No other app does this.

📈

Scrolling Stock Ticker

Real-time scrolling ticker with color-coded gains/losses. Add any symbol. CVDisplayLink-powered smooth animation.

💻

System Monitoring

CPU, RAM, GPU, temperatures, fan speeds, network, disk, battery, Bluetooth devices, top processes. With sparkline charts.

📝

Script Widgets

Full xbar/SwiftBar compatibility. Write shell scripts that render to the menu bar with colors, submenus, and SF Symbols.

📅

Calendar & Meetings

Mini calendar grid with event dots, next meeting countdown, one-click Zoom/Meet/Teams join links extracted from events.

⚡

Profiles

Save widget layouts as profiles. Quick-switch between Work, Home, Presentation, Developer setups with one click.

For Developers

Build your own widgets

Implement the BaristaWidget protocol, register it, done. Full access to AppKit, IOKit, EventKit, and any macOS API.

class MyWidget: BaristaWidget { static let widgetID = "my-widget" static let displayName = "My Widget" func render() -> WidgetDisplayMode { return .text("Hello World") } func buildDropdownMenu() -> NSMenu { ... } }

xbar Compatible

Drop in any xbar/SwiftBar plugin. The Script Widget parses the full xbar format - pipe params, submenus, colors, images.

#!/bin/bash
echo "Hello | color=green"
echo "---"
echo "Open Site | href=..."
echo "--Submenu | color=blue"

Native AppKit

Pure Swift, zero dependencies, no Electron, no web views. Compiles with swift build in seconds. ~18K lines total.

Swift 5.9 AppKit SPM IOKit EventKit CoreGraphics

Security Hardened

HTTPS-only network requests. Shell whitelist for script execution. URL-encoded API parameters. Sanitized inputs for all system commands. No auto-chmod. Blocked system path execution.
