For Developers

Extend it. Script it. Build on it.

Build your own widgets

Implement BaristaWidget, register it, done. Full access to AppKit, IOKit, EventKit, and every macOS API.

class MyWidget: BaristaWidget { static let widgetID = "my-widget" static let displayName = "My Widget" func render() -> WidgetDisplayMode { return .text("Hello World") } }

xbar Compatible

Drop in any xbar/SwiftBar plugin. Full format support: pipe params, submenus, colors, images, actions.

#!/bin/bash
echo "Hello | color=green"
echo "---"
echo "Open Site | href=..."
echo "--Submenu | color=blue"

Pure Native Stack

Swift + AppKit. No Electron, no web views, no frameworks. Compiles with swift build in under 5 seconds.

Swift 5.9 AppKit SPM IOKit EventKit CoreGraphics

Security Hardened

Open source and auditable. Every network request, script execution, and system call is locked down.

HTTPS enforced on all requests
Shell whitelist for script execution
Sanitized inputs on all commands
